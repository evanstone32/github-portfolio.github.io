Cabarrus Parks Discussion Board

This is a web application that is for posting dicussions/events for any parks in cabarrus county. The front end of this application utilizes html/css while the backed utilizes Javascript. All information about the parks is gathered using the 'openstreetmap' API.

Features

Login/Sign up

The user can choose to either create a new account or login to an already existing account. Without an account users can view posts and comments but cannot create any or RSVP to any events.

View/Search Parks

On the Home Page of the website all parks within cabarrus county are listed. Users can then use the search bar to find a specific park that they are looking for. Selecting these parks will bring the user to a page where they can view/post discussions or events for that park.

Create Posts/Events

On a specific parks page users are able to create a discussion post or an event post. Discussion posts are used for questions that a user may have about a park. While event posts are used to post activities that other users can rsvp to. Both types of posts allow for other users to comment so that they can answer questions or ask different questions questions.

Profile

On the profile page users can view information about their profile such as name, username, description, and date joined. On this page users can also choose to sign out so that they can login to a different account.
