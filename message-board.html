Computer Science Message Board

This web application provides a platform for users to discuss questions about the computer science field. Built with Pythons Flask for the backend, HTML/CSS and bootstap for the frontend and a MySQL database to store and manage user information, posts, and comments efficiently.

Features

Login/Sign up

The user can choose to either create a new account or login to an already existing account. This data was then stored/queried from the MySQL database that was linked to this site.

Profile Page

The user can view their information by navigating to the profile page. The info in this page is pulled from the database based on the currently logged in user.

Choose/Create Forum

User can choose from a list of forums that are created by previous users. If the user cannot find any forms that they like then they can choose to create a new one. Once a form is selected users can view, create, and comment on posts within that forum.

Create Posts/Comments

The user can choose a forum topic that they like and post a dicussion on said forum. Other users can see these posts and posts their own comments on them.
