Time Clock Project

This is a .NET application that utilizes C# in visual studio to create both the front and back end. This application allows for users to clock in/ clock out, view their time sheet, and manage employees.

Features

Login/Sign up

The user is allowed to either create a new account or login to an existing one. When creating an account a user can select if they are a manager or not. Managers have special privlages that allow them to manage lower level employees.

Clock in/Clock out

The user can choose whether or not to clock in or clock out. Their current clock status is displayed in the top right. This information is then updated in the time sheet.

Timesheet

This feature allows for users to view all of times that they have clocked in and out. They can also see a summary of the money that they have earned within the past week and the amount hours worked.

Manage Employees

This feature allows for managers to view all of the employees currently employeed by their company. They can then update their info such as pay rate, job description, and username.
